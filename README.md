The navigation bar includes links to different sections such as Chef, Menu, Login, and Signup. ### -> Furcreates
  FurCreates is an NGO website dedicated to helping stray dogs, designed using HTML, CSS, and Bootstrap with a compassionate and user-friendly layout. The homepage features a heartfelt quote expressing our vision, set against a background image of a stray dog to create emotional connection and awareness. The navigation bar includes a Donate page displaying a motivational quote and a Donate button that reveals a QR code for easy contributions. The Adopt page showcases various dogs as cards, each with an image and an Adopt button. When users choose to adopt, a simple form collects their details and address, enabling the organization to contact them directly. The website reflects the mission of FurCreates ‚Äî to promote kindness, adoption, and better lives for stray dogs through a clean and interactive web design.

### -> Todo using maven
  This is a To-Do List Web Application developed in Java using Maven, designed to manage daily tasks efficiently. The application provides a simple interface where users can add, update, delete, and refresh tasks easily. A text bar allows users to enter a task title, while a description box captures detailed information about each task. All tasks are displayed in a well-structured table, with a filter option that lets users view either completed or pending tasks. The project demonstrates effective use of Java, Maven, and web technologies to implement CRUD operations and dynamic task management with a clean and organized layout.
